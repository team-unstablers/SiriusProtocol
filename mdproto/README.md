# WAIT, WHERE IS THE `mdprotoc`?

MDProto Compiler (`mdprotoc`) is a tool that automates Protobuf and related code generation from MDProto message definition files (`.mdproto`, `.mdproto.md`).

However, `mdprotoc` is not yet included in this repository. `mdprotoc` and `protoc-gen-libsirius-cpp` will be released together when libsirius is open-sourced.
